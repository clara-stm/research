# CLARA Architecture Documents

This directory contains public architecture notes for the CLARA research repository.
The documents explain system responsibilities and design concepts at a level suitable
for public dissemination. They do not expose internal prompts, private datasets,
credentials, infrastructure identifiers, operational procedures, or non-public
implementation details.

## Documents

| Document | Summary |
| --- | --- |
| [AI_ROLES.md](AI_ROLES.md) | Explains the four main AI roles in the public CLARA architecture: CLARA Core, Audit Agent, Network Agent, and Browser / URL Risk Agent. |
| [EXECUTION_AND_INTEGRATION.md](EXECUTION_AND_INTEGRATION.md) | Reviews execution and integration architecture: framework/app-store packaging, Ratio1 execution, RedMesh partner integration, MISP, and Neo4j. |
