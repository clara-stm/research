# Documente de arhitectură CLARA

Acest director conține note publice de arhitectură pentru depozitul de cercetare
CLARA. Documentele explică responsabilități, concepte și moduri de colaborare la un
nivel potrivit pentru diseminare publică. Ele nu publică prompturi interne, seturi de
date private, credențiale, identificatori de infrastructură, proceduri operaționale
sau detalii de implementare nepublice.

## Documente

| Document | Sumar |
| --- | --- |
| [ROLURI_AI.md](ROLURI_AI.md) | Explică cele patru roluri AI principale din arhitectura publică CLARA: CLARA Core, Agentul de Audit, Agentul de Rețea și Agentul de Risc Browser / URL. |
| [EXECUTION_AND_INTEGRATION.md](EXECUTION_AND_INTEGRATION.md) | Analizează arhitectura de execuție și integrare: ambalare pentru cadre și magazine de aplicații, execuție pe Ratio1, integrare RedMesh, MISP și Neo4j. |
